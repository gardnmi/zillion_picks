About

What we do:

We provide college football picks for the Spread and Straight-Up bets. Every pick is updated daily until game day so you can be assured you have the most accurate pick possible.

How we do it:

Each day we scrape the web to get the most up to date data available. This information is then fed into our ensemble of machine learning models which in turn provides a non-biased prediction for each and every game.

Picks Table

The current week’s Picks Table is updated daily until game day. Most fields are self-explanatory but for those fields that may be not be so easily understood we've provided explanations below .

Field Definition
Sportsbook Spread (Avg) This is the average spread consensus from the most popular online sportsbooks. Negative number favors the home team and vice versa.
Regression Model Pick This is our regression algorithm prediction for the result of the game. Negative number favors the home team and vice versa
Regression Model Spread The regression algorithm predicts the final spread outcome to determine the pick.
Classification Model Pick This is our classification algorithm pick. It is an alternative model to the regression algorithm
Classification Model Pick The percent confidence of each pick from the classification algorithm. Regression algorithms do not provide confidence scores.
Final Game Result The actual outcome of the game
